  # Forest Client

Учебное приложение на JavaFX + MySQL для учета лесных участков.

## Возможности
- добавление участка
- просмотр списка участков
- редактирование и удаление
- назначение лесников
- хранение данных в MySQL

## Запуск
1. Запустить MySQL
2. Создать базу `forest_db`
3. Выполнить SQL для таблиц
4. Запустить команду:

```bash
mvn clean javafx:run
